# Python QR Code Generator

A simple Python script that generates a QR code from a user-provided URL and saves it as an image.

## Features

- Accepts user input for URL
- Custom filename support
- Error handling
- Saves QR code as JPG

## Requirements

- Python 3.x
- qrcode library

Install dependency:

```bash
pip install qrcode[pil]
```

## How to Run

```bash
python qr_generator.py
```

## Example Use

- Generate QR codes for websites
- Share links easily
- Learning Python libraries
